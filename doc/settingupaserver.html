Flight 2942 Documentation

Setting up a Server

Requirements:

  • PHP
  • MySQL
  • A web host (apache, lynx, whatever)

Performance off of the default server (http://cupm.net/public/bacongamejam-01/server.php) is very slow because it is from a BlueHost server. To increase performance, run it localhost on your machine. I've noticed the best performance happens when it is run over a network.

Provided is steps to installing a Flight 2942 server.

  1. Upload server files.

    2. Copy server.php and server-config.php to a public directory. In this case, we will use /var/www/.

  2. Create a MySQL database and user.

    3. In this case, the database name will be bacongamejam-01

    Note: Since this software was made so quickly, I highly suggest you make a new user for this software, as I have not done any injection testing or protection.

  3. Import the database structure.

    4. Import or recreate the structure as defined by structure.sql

  4. Configure the server.

    5. Configure server-config.php to properly connect the MySQL database.

    Note: An easy way to check that the server is working is to visit the page via a browser, and see if the page returns "{}". If so, your server is properly set up. If not, an error message will show up.

  5. Configure the client to connect to the server.

    6. Currently, the game client reads from the server.lua file and uses that as the server. The cupm server connection (located in server-cupm.lua) looks like this:

      server.name = "cupm.net (Seppi)"
  server.url = "http://cupm.net/public/bacongamejam-01/server.php"

    An example of connecting to the localhost (located in server-localhost.luawould be like this:

      server.name = "localhost"
  server.url = "http://localhost/bacongamejam-01/server.php"
    To set up server.lua for your server, you will need to name the server (server.name) and give it your url (server.url). The best way is to find your outbound IP address, and use that. e.g. : 
      server.name = "Foo's Internal Server"
  server.url = "http://192.168.1.2/bacongamejam-01/server.php"
    To find your ip address, open up a terminal and type ifconfig.
    To find your ip address, open up a terminal and type ipconfig.

  6. Play Flight 2942

    7. And that's all folks! Have fun. Just connect, and go.
